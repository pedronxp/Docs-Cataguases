#Implementação concluída! Aqui está o resumo do que foi feito:

**✅ Menu Lateral**

- Adicionados **Decretos** (com sub-itens Listar/Novo) e **Memorandos** no grupo "Fluxo de Trabalho"
- Adicionado **Gerenciar Feedbacks** na seção Administração (visível só para ADMIN_GERAL)

**✅ Novas Páginas**

- `/administrativo/decretos` — Listagem completa de decretos (reutiliza o mesmo hook/componentes das portarias)
- `/administrativo/decretos/novo` — Página de criação
- `/administrativo/memorandos` — Tela "Em Construção" com design premium
- `/admin/feedbacks` — Painel admin para ver, filtrar e **responder** feedbacks

**✅ Backend**

- `GET /api/feedback` — Admin lista todos os feedbacks
- `PATCH /api/feedback/[id]` — Admin responde e atualiza status
- Campo `resposta` adicionado ao `Feedback` no banco via `prisma db push`

**⚠️ Nota importante** Após iniciar o `npm run dev`, o TanStack Router vai regenerar o arquivo de tipos e os erros de lint nas novas rotas vão desaparecer automaticamente.
